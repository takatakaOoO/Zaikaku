# プライバシーポリシー

**最終更新日: 2026年2月1日**

本プライバシーポリシーは、「材確（Zaikaku）」（以下、「本アプリ」といいます）における利用者情報の取り扱いについて定めたものです。

## 1. 収集する情報とその利用目的

本アプリは、サービスの提供および向上のために、以下の情報を収集・利用する場合があります。

### 1.1 端末情報・広告識別子
本アプリでは、広告配信のために **Google AdMob** を利用しています。AdMobは、広告の最適化や効果測定のために、端末情報や広告ID（Advertising ID）などの情報を自動的に取得する場合があります。

詳細については、Googleのポリシーをご確認ください：
- [Google 広告 – ポリシーと規約](https://policies.google.com/technologies/ads?hl=ja)

### 1.2 カメラおよび画像データ
本アプリは、バーコードのスキャンおよび文字認識（OCR）のために、デバイスのカメラおよび画像ギャラリーへのアクセスを使用します。
- 撮影された画像データやスキャンされたバーコード情報は、アプリ内での照合・登録処理にのみ使用され、**外部サーバーへ送信・保存されることはありません**。すべて端末内で処理されます。

### 1.3 利用履歴データ
アプリ内でスキャンされた履歴データ（時刻、判定結果、バーコード値）は、利用者の端末内部（ローカルデータベース）にのみ保存されます。ユーザー自身が「エクスポート」機能を使用して外部（メール等）へ送信しない限り、開発者や第三者がこれらのデータを閲覧することはありません。

## 2. 第三者への提供
本アプリは、以下の場合を除き、利用者の同意なく個人情報を第三者に提供することはありません。
- 法令に基づく場合
- 広告配信事業者（Google AdMob）によるデータ収集（上記1.1参照）

## 4. プライバシーポリシーの変更
本ポリシーの内容は、法令改正やアプリの機能変更に伴い、予告なく変更されることがあります。重要な変更がある場合は、アプリ内または本ページにて通知します。
