# 材確 (Zaikaku) 取扱説明書

## 1. はじめに
「材確（ザイカク）」は、製造現場における材料の取り間違いを防止するためのバーコード照合アプリです。
製造指示書（製品）と現品（材料）のバーコードをスキャンし、正しい組み合わせかどうかを瞬時に判定します。

## 2. 基本的な使い方

### 2.1 トップ画面
アプリを起動するとトップ画面が表示されます。
- **スキャン開始**: バーコード照合を開始します。
- **製品マスタ**: 照合に使用する製品情報の登録・編集を行います。
- **履歴閲覧**: 過去のスキャン結果の履歴を確認できます。

### 2.2 製品の登録（準備）
照合を行う前に、製品情報と使用する材料を登録する必要があります。

1. トップ画面から「製品マスタ」をタップします。
2. 右下の「＋」ボタンをタップして新規登録画面を開きます。
3. **製品名**と**型番**を入力します（OCRボタンでカメラ入力も可能です）。
4. **材料バーコード**を登録します。
    - **手入力**: バーコードの数値を直接入力します。
    - **画像スキャン**: 「画像から追加」をタップし、指示書の写真などを選択すると、写っているバーコードを一括で取り込めます。
5. 「保存」ボタンをタップして完了です。

## 3. 照合フロー

### 3.1 照合開始
1. 「スキャン開始」ボタンをタップします。
2. カメラが起動します。画面上部のプルダウンから、これから作業する**製品**を選択してください。
    - 選択した製品に登録されている材料のみが「OK」となります。

### 3.2 スキャンと判定
1. 画面中央のガイド枠にバーコードを合わせます。
2. 読み取りに成功すると、画面が一時停止し結果が表示されます。
    - **OK（緑）**: 正しい材料です。
    - **NG（赤）**: 間違った材料、または登録されていない材料です。
3. 結果を確認し、**シャッターボタン**（または画面タップ）で次のスキャンへ進みます。
4. 画面下部のリストには、その製品に必要な材料一覧が表示され、照合済みのものにはチェックがつきます。

## 4. 履歴と出力

### 4.1 履歴の確認
「履歴閲覧」画面では、過去の全てのスキャン結果がリスト表示されます。
- OK/NGの結果、スキャン時刻、対象の製品名が確認できます。

### 4.2 データの書き出し
履歴データをCSVファイルとしてメール送信などに利用できます。
1. 履歴画面右上の「共有」アイコンをタップします。
2. 対象期間（開始日・終了日）を選択し「エクスポート」をタップします。
3. スマートフォンの共有メニューが開くので、メールアプリ等を選択して送信してください。

## 5. よくある質問 (FAQ)

**Q. スキャンしても反応しない**
A. カメラのレンズが汚れていないか確認してください。また、バーコードが破損している場合や、照明が暗すぎる場合は読み取れないことがあります。

**Q. 「未登録のバーコード」と表示される**
A. 選択中の製品マスタに含まれていないバーコードを読み取っています。製品マスタの内容を確認してください。

**Q. 広告が表示される**
A. 本アプリは無料版のため、画面下部に広告が表示されます。機能に制限はありません。
