## 重要な注意事項

### プライバシーポリシーURL

ストア登録には、**公開可能なプライバシーポリシーURL**が必須です。

**選択肢**:
1. **GitHub Pages**: 既存の`docs/PRIVACY_POLICY.md`をGitHub Pagesで公開
2. **自社サイト**: 既にお持ちの場合
3. **無料ホスティング**: Google Sites等

この点について、どの方法を採用されるかご検討ください。

### AdMob App ID

リリースビルド時には、**本番用AdMob App ID**を`key.properties`に設定し、以下のコマンドでビルドする必要があります:

```bash
flutter build appbundle --release --dart-define=ADMOB_BANNER_ID_ANDROID=ca-app-pub-XXXXXXXX/YYYYYYYYYY
```

テスト用IDのままアップロードしないようご注意ください。
