## 詳しい説明（4000文字以内）

```
【材確（ザイカク）- 製造現場の材料照合を効率化】

材確は、製造ラインでの材料ピッキング作業を支援する、現場特化型のバーコード照合アプリです。
製造指示書に基づいた材料の正誤判定を瞬時に行い、音声フィードバックで作業者に即座に結果を伝えます。

■ こんな課題を解決します
・材料の取り違えによる製造ミスを防止したい
・目視確認だけでは不安、確実な照合手段が欲しい
・作業履歴を記録して、後から確認できるようにしたい
・紙の指示書管理が煩雑で、デジタル化したい

■ 主な機能
【バーコードスキャン】
カメラで材料のバーコードを読み取り、製品マスタと照合します。
1D（JAN、CODE128等）、2D（QRコード）の両方に対応。

【音声フィードバック】
正解時は「ピンポン」、不正解時は「ブー」の音で即座に判定結果を通知。
画面を見なくても作業を進められます。

【製品マスタ管理】
製品ごとに必要な材料バーコードを事前登録。
OCR機能で製品名・型番をカメラから入力でき、登録作業も効率的です。

【スキャン履歴】
いつ、どの材料をスキャンしたか、結果はどうだったかを自動記録。
日付範囲を指定してCSVエクスポートし、メールで共有できます。

【待機モード】
誤スキャンを防ぐため、結果確認後に「次へ」ボタンを押すまで次のスキャンをブロック。
慎重な作業が求められる現場に最適です。

■ 使い方
1. ホーム画面で「スキャン開始」をタップ
2. 製品を選択（事前に製品マスタに登録）
3. 材料のバーコードにカメラをかざし、シャッターボタンを押す
4. 音声と画面で結果を確認
5. 「確認」ボタンで次の材料へ

■ こんな現場におすすめ
・製造業の組立ライン
・部品ピッキング作業
・検品・検査工程
・倉庫での出荷準備

■ 動作環境
・Android 5.0以上
・カメラ機能必須
・インターネット接続不要（広告表示時のみ使用）

■ プライバシーとセキュリティ
・全てのデータはデバイス内に保存され、外部サーバーへの送信は一切ありません
・カメラはバーコードスキャン時のみ使用します
・詳細は「プライバシーポリシー」をご確認ください

■ サポート
ご不明な点やご要望がございましたら、アプリ内の「取扱説明書」をご覧いただくか、
サポートメールまでお問い合わせください。

製造現場の品質向上と作業効率化を、材確がサポートします。
```

## スクリーンショット推奨構成

1. **ホーム画面**: アプリの第一印象、シンプルで分かりやすいUI
2. **スキャン画面**: メイン機能、カメラビューとガイド枠
3. **判定結果表示**: 正解時の画面（緑のチェックマーク、材料リスト）
4. **履歴画面**: スキャン履歴の一覧表示
5. **製品マスタ画面**: 製品一覧または登録画面
6. **OCRスキャン画面**（オプション）: 差別化ポイントとして

各スクリーンショットには、機能を説明する短いキャプションを追加することを推奨します。
